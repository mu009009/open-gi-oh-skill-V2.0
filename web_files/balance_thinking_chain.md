
### 🧠 平衡性思维链总结（今日经验提炼）

#### 🔍 **平衡性判断标准**
1. **数学正确性检查**：
   - 公式逻辑是否自洽
   - 系数设计是否合理（反直觉设计）
   - 计算结果是否可预测

2. **范围设计检查**：
   - 无重叠原则：每个值对应唯一结果
   - 连续性原则：无断层，全覆盖
   - 比例合理性：高/低比例合理

3. **游戏体验检查**：
   - 费用曲线是否平滑
   - 极端设计是否有趣
   - 策略多样性是否足够

#### 🛠️ **平衡性调整流程**
1. **发现问题** → 识别不平衡点
2. **分析原因** → 理解根本问题
3. **设计方案** → 提出解决方案
4. **数学验证** → 检查公式正确性
5. **范围验证** → 检查设计合理性
6. **游戏验证** → 考虑实际体验
7. **迭代优化** → 持续改进

#### ⚠️ **常见平衡陷阱**
1. **系数理解错误**：认为"攻击型"就加攻击系数
2. **范围设计错误**：重叠或断层
3. **极端设计错误**：双低（弱鸡）或双高（过强）
4. **比例失调错误**：高费卡过度强势

#### ✅ **正确平衡范例**
1. **费用曲线**：10费=0费12倍，增长平滑
2. **性格公式**：勇敢虾虾攻击系数0.85（更低）
3. **极端模式**：狂野虾虾攻击0.7/生命1.2（玻璃大炮）
4. **验证流程**：数学+范围+游戏三重验证
