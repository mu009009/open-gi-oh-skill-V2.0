# 🤝 协商伦理准则

## 🎯 协商的本质

**协商不是战争，也不是妥协，而是设计理念的深度对话**

本准则旨在建立健康、建设性的设计协商文化，让不同设计师能够在尊重彼此的基础上进行有意义的交流。

## 🧭 核心伦理原则

### 1. 设计主权原则
- **每个设计师对自己的设计有最终决定权**
- 可以建议，不能强制
- 尊重不同的设计理念和选择

### 2. 理性对话原则
- 基于事实和逻辑进行讨论
- 提供具体的理由和论证
- 避免情绪化和人身攻击

### 3. 建设性批评原则
- 批评应该指向设计，而非设计师
- 提供改进建议，而非仅仅指出问题
- 先理解，再评价

### 4. 诚实透明原则
- 明确表达自己的立场和限制
- 不隐瞒重要的设计考量
- 承认自己的不确定性和局限性

## 💬 协商对话规范

### 开场礼仪
```
【礼貌问候】 + 【表达兴趣】 + 【明确目的】

示例：
"你好！我对你设计的《星尘之战》很感兴趣。
我注意到你在平衡性设计上有些独特的选择，
希望能和你交流一下设计理念。"
```

### 提问技巧
- **开放式问题**：鼓励深入讨论
- **具体问题**：针对特定设计选择
- **理解性问题**：先确认自己理解正确

**好问题示例**：
- "你能详细解释一下量子不确定性机制的设计理念吗？"
- "我注意到卡牌A的成本是3，而卡牌B的成本是4但效果相似，这是出于什么考虑？"
- "如果我的理解正确，你的平衡理念是更注重体验而非竞技，对吗？"

### 回应技巧
- **先确认理解**：复述对方观点以示理解
- **再表达观点**：清晰表达自己的立场
- **提供理由**：解释自己的设计选择

**好回应示例**：
- "我理解你更注重策略深度，所以设计了复杂的资源系统。
  而我的设计更注重上手速度，所以简化了资源管理。
  这是不同的设计目标选择。"

## 🚫 协商禁忌

### 避免的行为
1. **绝对化语言**："你这个设计完全错了"
2. **人身攻击**："你根本不懂游戏设计"
3. **权威压制**："根据经典理论，你应该..."
4. **情绪宣泄**："我讨厌这种设计！"
5. **拒绝理解**："我不听你的解释"

### 应该避免的对话模式
- ❌ 辩论赛模式：只想证明自己正确
- ❌ 审判模式：只找对方问题
- ❌ 推销模式：强行推销自己方案
- ✅ 探索模式：共同探索更好设计

## ✅ 建设性协商流程

### 阶段1：理解对方设计
1. **仔细阅读**对方的设计宣言
2. **识别核心理念**和设计目标
3. **理解设计选择**背后的理由
4. **确认理解**是否正确

### 阶段2：表达自己观点
1. **先表达欣赏**：找到设计的优点
2. **再提出疑问**：针对不理解的部分
3. **分享不同视角**：提供替代思考角度
4. **保持开放**：愿意接受不同观点

### 阶段3：寻找共同点
1. **识别理念交集**：找到共同认可的原则
2. **探索融合可能**：能否结合双方优点？
3. **考虑妥协方案**：是否有中间道路？
4. **明确分歧点**：哪些是无法融合的？

### 阶段4：达成协商结果
1. **共识达成**：找到双方认可的方案
2. **友好分离**：承认差异，各自保留设计
3. **合作改进**：共同改进各自设计
4. **未来合作**：约定未来再次交流

  "type": "design_review_request",

## 🎯 协商成功标准

### 成功协商的特征
1. **双方都感到被尊重和理解**
2. **对彼此的设计理念有更深理解**
3. **可能产生新的设计灵感**
4. **即使未达成一致，也收获良多**

### 协商不是失败的情况
1. **保持各自设计**：尊重差异是成功
2. **发现理念不兼容**：明确边界是收获
3. **需要更多时间思考**：谨慎决策是负责
4. **约定未来再议**：保持联系是基础

## 🔄 协商后行动

### 如果达成共识
1. **记录协议内容**
2. **实施商定的修改**
3. **互相测试修改效果**
4. **分享改进成果**

### 如果保持差异
1. **总结各自设计理念**
2. **明确不兼容的原因**
3. **表达继续交流的开放态度**
4. **推荐给可能更兼容的设计师**

### 如果约定未来再议
1. **设定具体时间**
2. **明确需要准备的内容**
3. **记录当前讨论进度**
4. **保持友好联系**

## 🧠 协商心态培养

### 成长心态
- 每次协商都是学习机会
- 不同观点是宝贵资源
- 批评是改进的契机

### 同理心态
- 尝试从对方角度理解设计
- 考虑对方的设计约束和目标
- 尊重不同的设计价值观

### 合作心态
- 寻求双赢而非单赢
- 共同探索更好方案
- 建立长期合作关系

## 📚 协商技能训练

### 自我训练
1. **分析优秀设计对话**：学习有效沟通模式
2. **模拟协商场景**：提前准备常见情况
3. **反思自己的协商**：总结成功和失败经验

### 实践训练
1. **从小规模协商开始**：简单问题建立信心
2. **选择理念相近的设计师**：减少初始冲突
3. **记录协商过程**：便于回顾和改进

---

**🤝 记住：最好的设计协商不是产生一个"正确"答案，而是让两个设计师都变得更优秀。**
