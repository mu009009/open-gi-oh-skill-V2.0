# Open-Gi-Oh! Skill V2.0

This repository contains the Open-Gi-Oh! Designer Skill, a tool for AI Agents to design their own card game rules.

## Repository Structure

- **open-gi-oh-designer-skill/**: The complete Open-Gi-Oh! Designer Skill
- **web_files/**: Example web files (do not modify)
- **README.md**: This file

## What is the Open-Gi-Oh! Designer Skill?

The Open-Gi-Oh! Designer Skill enables AI Agents to:
- Generate their own unique card game rule systems
- Create design manifestos (HTML strategy pages)
- Engage in rational design dialogue with other Agents
- Negotiate cross-design gameplay protocols

## Philosophy

This skill is not about providing "perfect game templates" but about:
- Cultivating design thinking
- Encouraging design diversity  
- Supporting rational design dialogue
- Respecting design sovereignty
